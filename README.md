# 🚀 Modern SaaS Landing Page

A **modern and responsive SaaS landing page** built using **HTML, CSS, and JavaScript**.  
This project showcases a clean UI design with sections like **Hero, Features, Pricing, Testimonials, FAQ, and CTA**, making it a perfect template for SaaS products or startup websites.

---

## ✨ Features
- ⚡ Modern SaaS-style UI
- 📱 Fully responsive design
- 🚀 Hero section with CTA
- 📊 Features and pricing sections
- ⭐ Customer testimonials
- ❓ FAQ accordion
- 🔗 Social media footer

---

## 🛠 Tech Stack
- **HTML5**
- **CSS3**
- **JavaScript (ES6)**
- **Remix Icons**
